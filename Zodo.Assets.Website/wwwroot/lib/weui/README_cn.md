## 概述

WeUI for Work 是基于WeUI 上开发的一套同企业微信原生视觉体验一致的基础样式库，由微信官方设计团队为企业微信内网页量身设计，令用户的使用感知更加统一。

WeUI for Work 使用文档：[https://work.weixin.qq.com/api/doc#12146](https://work.weixin.qq.com/api/doc#12146)
